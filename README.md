## Film App

To clone and run this application, run this from your command line with Git and Yarn or NPM installed.

```bash
# Clone the repository
$ git clone https://github.com/DomMoorcroft/film-app.git

# Go to the downloaded repository
$ cd film-app

# Install the depenencies with Yarn/NPM
$ yarn installl
or
$ npm install

# Run the application
$ yarn start
or
$ npm start
```
